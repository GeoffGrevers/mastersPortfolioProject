Pattern Manipulation Visualisation


Instructions:
1. Select notes on the sequencer
2. Select the manipulations you would like to use
3. Press play
Please pause before changing manipulations, or it may not work!

By default manipulation 1 is rotate anticlockwise, while manipulation 2 is interpolation.
Click on a saved pattern to load it.

Libraries used to make this website:
Tone.js (music/sounds)
p5.js (graphics)
dat.gui.js (graphical user interface (the menu top right))

The purpose of the system is to allow for musicians with a beginning curiosity into algorithmic music to have immediate access to a system that can be used to visualise organised sounds composed using algorithmic techniques, with a small a barrier to entry as possible. Using a step sequencer, it follows standards that are present in popular music making software, which allows for one to use the system without having to learn how to. In the current landscape of algorithmic music there are tools available to the people deeply involved within the field of study in the form of coding languages (for example ixi lang or TidalCycles), while there are also music making games (such as blockdust) based on algorithmic principles, with no representation of the field of study. This system is a higher-level abstraction compared to coding for music, shown visually.

Behaviour of the manipulations:
Rotate (prime and retrograde): This cycles through the sequencer, with the next run through shifting the position of the notes either fowards or backwards.
Interpolate: This assigns notes randomly to columns that have no notes yet.
Reverse: Reverses the notes that have been input.

The following are pattern manipulations that are built into the tone.js library:
up - cycles upward
down - cycles downward
upDown - up then and down
downUp - cycles down then and up
alternateUp - jump up two and down one
alternateDown - jump down two and up one
random - randomly select an index
randomWalk - randomly moves one index away from the current position
randomOnce - randomly select an index without repeating until all values have been chosen
